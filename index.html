✦ Regole grammaticali ▾

Struttura della frase

  • Nessuno spazio tra le parole
  • Separatore tra termini: '
  • Pausa breve: :n:
  • Pausa marcata: :nn:
  • Pausa rituale: :nar:

Genere e numero

  • Maschile: -morg
  • Femminile: -shal
  • Plurale: -zhar
  • Es: Ssha-morg-zhar = servi maschi

Tempi verbali

  • Presente → nessun suffisso
  • Passato → suffisso -vrath
  • Futuro → suffisso -thruk

Es: uccidere → Drakth

uccise → Drakth-vrath

ucciderà → Drakth-thruk

Es: essere → Zovak

era → Zovak-vrath

sarà → Zovak-thruk

Negazione

  • "non" / "no" → suffisso -nux sulla parola seguente
  • Es: non morire → Dravth-nux
  • Es: non tradire → Drakthul-nux
  • Es: io non sono → Vok'Driv-nux

Pronomi

  • Io / me / mi → Vok
  • Tu / te / ti → Zik
  • Lui / lei → Thul (masch: Thul-morg, femm: Thul-shal)
  • Noi / ci → Ruk
  • Voi / vi → Skul
  • Loro / li / le → Drak

Numerazione

Prefisso Kh- + cifre in lettere separate da ', racchiuse tra :n:

  • 1=Velu 2=Gruz 3=Thal
  • 4=Zuvu 5=Drin 6=Sorith
  • 7=Brak 8=Vulth 9=Xirn

Es: 1723 → :n:Kh-Velu'Brak'Gruz'Thal:n:
